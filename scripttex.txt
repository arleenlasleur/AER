1. Launch clean Unrealed (remove any EditPackages=)
2. Textures -> File -> Import
   aerscreenbg             (this is basetexture aka bg texture of scriptedtexture)
   Package = aertex        (name of .utx file that will be imported in main .u file)
   Group   = Disp          (what it called in similar textures inside of .u file)
   Name    = aerscreenbg   (conventional name)
   GenMipMaps  = checked   (NFI of that. used for LOD)
   Compression = BC3/DXT5
3. Save .utx
   File -> Save -> unrealroot\textures
4. File -> New
   Class = ScriptedTexture
   Size  = 256x256
   Name  = aerscreen
   Group = Disp
5. Set settings
   ScriptedTexture\bUse32BitFormat = true
   SourceTexture = aerscreenbg
6. Save .utx
7. Move .utx file to work_pkg_dir: unrealroot\textures -> unrealroot\aer\textures
8. #exec obj load file="textures\aertex.utx" package="AER"
   (package = NOT name of .utx file, IS name of future .u file)
9. Rebuild
   (uncomment EditPackages=, ucc make)
