Metric system ruler converter for unrealed, v. 0.2-2025.01.04 by Arleen Lasleur.
Press Enter second time after math expression parsed, to invoke convert again.
For imperial or other systems you can change uu_to_m variable.

UU
@UU grid
Hotkeys:
Bksp/Del/arrows - normal char undo
Numpad/Ent      - enter data
Y U             - switch to metric/unrealed units input mode
Unsupported while working in UU -> metric mode
Ins/Shift+Del   - reset/clear
+ - / *         - unrealed-style math
PgUp/Dn         - snapped roll
C M             - set scale to cm / m
Q W E R         - set grid to 4 / 8 / 16 / 32
I               - remove non-integer part of input
